## Package manager

### yarn

```sh
npm install -g yarn
```

## Project setup

```sh
yarn
```

## Compiles and hot-reloads for development

```sh
yarn run dev 또는 npm run dev
```

## Compiles and minifies for production

```sh
yarn build 또는 npm run build
```
